query_tcga
===============================

author: Jacki Novik

Overview
--------

Helper functions to download TCGA data from GDC

Installation / Usage
--------------------

To install use pip:

    $ pip install query_tcga


Or clone the repo:

    $ git clone https://github.com/jburos/query_tcga.git
    $ python setup.py install
